# Requirements

Bu repo bir **npm workspaces** monorepo’sudur.

## Gerekli

- Node.js (önerilen: LTS)
- npm (Node ile gelen)
- Docker + Docker Compose v2

## Opsiyonel (modüle göre)

- **OMR (OpenCV)**: `docs/OMR_SETUP.md`
- **Desktop (Electron)**: macOS/Windows/Linux (Electron destekli)
- **Mobile (Expo)**: Xcode (iOS), Android Studio/SDK (Android)
